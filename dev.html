Contributing to TripoliÂ¶

For development, it is recommended to install the Python modules in editable form.

pip install --user --editable .

To run Emacs with Tripoli, you can use the make run target.

Run the tests with pytest using the make check target or the make check-verbose target.

You can run Emacs with the built library using the make run target or the make run-bare target (which does not run your init).

You can enter a REPL the using make repl target. When starting a REPL, the module emacs_raw is automatically imported as er.

Note

So far I have not been able to embed IPython or anything more sophisticated than the default Python REPL (help appreciated).
